# CODSOFT Python Programming Internship

This repository contains tasks completed as part of the CodSoft Python Programming Internship.

## Task 1: To-Do List Application
- Developed using Python
- Menu-driven command-line application
- Features:
  - Add tasks
  - View tasks
  - Delete tasks
- Concepts used: Lists, loops, conditional statements, user input handling

## Task 2: Password Generator
- Developed using Python
- Generates secure random passwords
- Allows user to select:
  - Password length
  - Character types (uppercase, lowercase, numbers, symbols)
- Includes password strength evaluation
- Concepts used: Functions, random and string modules, exception handling
